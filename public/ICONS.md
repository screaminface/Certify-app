# PWA Icon Placeholders

For production, replace these placeholder files with proper icons:

1. Generate icons using a tool like:
   - https://www.pwabuilder.com/imageGenerator
   - https://github.com/onderceylan/pwa-asset-generator
   - https://realfavicongenerator.net/

2. Required sizes:
   - pwa-192x192.png (192x192 pixels)
   - pwa-512x512.png (512x512 pixels)  
   - apple-touch-icon.png (180x180 pixels)
   - favicon.ico (32x32 pixels)

3. Place generated icons in this /public folder

4. Icons should have:
   - Transparent or solid background
   - Clear, simple logo/icon
   - Good contrast for visibility

The placeholder vite.svg can be replaced with your custom icon design.
